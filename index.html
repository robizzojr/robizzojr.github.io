Exponent Rules: The Essentials


Exponent rules show up a lot on the ACT and SAT.

However, many students get confused by operations involving them.

Recall the following rules:


Addition and Subtraction:

a2+ a2 = 2a2

"a" is our base. If two bases are the same and have the same exponent, We can combine them.

We can prove this using numbers instead of variables:


23 + 23 = 2(23) = 16

However, we cannot combine bases with different exponents or different bases featuring the same exponent :


b3 + b4 ≠ 2b7      and      x4- y4 ≠ -xy4

Multiplication and Division:

x4·x 3 = x7


For multiplying, when two variables share the same base , we can simply re-write them as one base and add the exponents.

Notice, the exponents do not have to be the same.


y5⁄y 2 = y3


For division, if bases are the same, simply subtract exponents and write as one base.





x2·y4 ≠ xy6


Here, the bases are not equal. Nothing can be done!

5x2y · 3 x5y7 z4 = 15 x7y8 z4


Even though the second term has a Z that the first one doesn't,

we can still "combine" the variables that do have the same base.
